# 問3の回答
問3は小問題が2つあるので，1つずつ説明いたします．
### 問a.の回答
```
以下に私が想像した通信の様相，インターネット全体の通信の様相，それらの違いについて，順に理由を交えながら述べる．

【私が想像した通信の様相】
　私は現状の通信の様相として，スマートフォンやIoT技術搭載の製品に対して特化し始めていると考えた．
理由として，総務省の調査によると，2017年の各世帯における情報通信機器の保有率は，パソコンが72.5%，モバイル端末が94.8%と多くの人が固定デバイスよりもモバイルデバイスを有し，
サービスプロバイダもそれに応じて固定デバイスよりの通信（有線LANなど）からモバイルデバイスに適した通信（無線LAN，Wi-Fiなど）に力を注いでいるのではないか，と考えたためである．
また，IoT機器に関しても同様のことが言える．以下に詳しく説明する．

　まず，トラフィックについて考える．
これに関しては，固定デバイスのトラフィックの成長率は，モバイルデバイスのトラフィックの成長率より下回っていると考えられる．
理由として私は２つ考えた．
1つ目は，スマホの需要の高さである．
最近では，学校や仕事，家庭内で連絡を行う際は，LineなどのSNSを使用することがほとんどである．
理由として，固定デバイスと違いどこででも連絡が可能であり，電話のように時間がかからないからであると考えられる．
また，スマホには翻訳アプリ，地図アプリ，写真アプリなど，数々な便利な機能を搭載しているため，これもスマホの需要を高める要因となる．
2つ目は，動画のストリーミング再生が増えていることが考えられる．
若者の間では，YouTubeやTikTokの人気が上昇している．
確かにYouTubeなどはパソコンなどで見られるが，TikTokなどはスマホなどのアプリからしか見られない．
また，どこででも見られるということもあり，多くの人は，スマホで動画を見るだろう．
また動画のトラフィックは，電話やメールなどに比べて圧倒的に大きい．
これもトラフィックを上げる原因の一つである．

　次に，通信速度について考える．
近年は，4Gなどの高速な無線通信システムが全国で整備され，さらには4Gに続く次世代モバイル通信技術として，5Gが実用化に向けられている，といったニュースがよく見られる．
このように通信事業者は無線通信に力を入れていることが分かる．
つまり，現在の通信の様相として，無線通信の速度の向上が著しく，有線通信の速度に関してはあまり変化が無いのではないか，と私は考える．

　次に，デバイスの数について考える．
小学生でもスマホを持つようになっているこの時代では，数年前と比べて，世界的規模で通信デバイスの数が増大していると推測できる．
私の考えでは，一人当たりの通信デバイス所持数は，1.5台程度と考える（一家に一台パソコン，また一人一人がスマホを持っていると考える）．
ここで考えられる問題としては，IPアドレスの枯渇と，通信速度の問題である．
CSMA/CD方式などで通信を行う場合，デバイス数が増えればその分コリジョンが増加し，通信速度が遅くなると考えられる．
この問題に関しては，5Gなどの技術でカバーできるかもしれないが，IPアドレスの枯渇化は，問題であると考える．
確かに，IPv6という技術はあるが，現在はIPv4が主流に使われている．
現在の通信の様相として，IPv6に移行させている途中段階であると考えられる．

　最後に，セキュリティについて考える．
現在はインターネットに詳しくない人でもスマートフォンなどからインターネットを利用できる．
私は，この現状を利用してフィッシング詐欺などが増えているのではないかと考える．
電話を使ったオレオレ詐欺のような手法に代わり，インターネット上でクレジットカードの暗証番号などを入力させるような手法が，増加していると考える．


【インターネット全体の通信の様相】
　最新のVNI（Cisco Visual Networking Index）の予測によると，2017年から2022年までに，全世界のIPトラフィックは3倍近くまで増大すると予想されている．
これは，全体に対するモバイルトラフィックの割合が増加していることに関係するが，固定トラフィックとモバイルトラフィックの増加は拮抗している国が多い．
また，一人当たりのデバイス数は増加傾向にあり，M2M，スマートフォンの順に成長率が高い．
この影響もIPトラフィックの増大に貢献している．
M2Mは一般家庭にも幅広く浸透されることが予測されており，トラフィックは2017年から2022年には7倍になると予想されている．
また，ビデオデバイスもトラフィックに多大な影響を与えており，全ての形式のIPビデオは2022年にはトラフィック全体の82%を占めると予測されている．

　次に，IPアドレスについてだが，現状IPv4は世界中の多くの地域ですでに枯渇しており，IPv6への移行は重要視されている．
全世界のモバイルネットワーク事業者は，IPv6を幅広く導入し，多くの量のIPv6トラフィックをサポートするようになった．
将来的に，トラフィック全体のかなりの部分がIPv6により生成される可能性があり，IoTを利用することを求めるプロバイダにとって注目されている．

　次に，コードカッティングについて述べる．
従来型のテレビ視聴は，オンラインビデオなどのインターネットによるビデオ視聴に取って代わられると予測されている．
従来型のテレビを利用する世帯よりも，インターネットテレビを利用している世帯の方が1.7倍程トラフィックを生成している．
原因として，インターネットテレビは多くのトラフィックを生成することが挙げられる．
なおビデオアプリケーションが要求する帯域幅は，現在も増加傾向にある．

　次に，セキュリティに関して述べる．
2018年9月時点で864件，約3420万件のデータ漏洩があり，1レコードあたりのコストは増加傾向にある．
通信においては，2017年から2018年にかけてDDoS攻撃が増加傾向にあり，そのトラフィックは全体の25%を占めることもあると言われている．
ブロードバンドの平均速度は高速化が続いており，それによりIPトラフィックも上昇している．
またモバイルネットワークの平均接続速度も向上しており，2017年から2年間で約2倍向上している．
また，全体のモバイルデバイスの利用率も増大している．Wi-Fiの通信速度も向上しており，ホットスポットの需要も高まっている．
それに伴い，802.11axなどの高速なワイヤレス回線も順々に実装されている．


【違い】
　まずトラフィックだが，私の予想に反して固定トラフィックの増加がモバイルトラフィックの増加に拮抗している国が増加していることが分かった．
理由として，スマホなどのモバイルデバイスのアプリケーションが充実して，モバイルトラフィックも増加しているのだろうが，
それ以上にPCなどの固定デバイスのアプリケーションが発展していると考えた．
スマホで動画が見られるとはいえ，PCで見た方が画面が大きく見やすい．
また解像度が上がる分データ量も増加するため，トラフィックの増加に貢献しているのであろう．
また，企業や学校においても，作業の効率化のためにPCを使う場所が増えていると考えられる．

　次に通信速度についてだが，固定ブロードバンドの速度はあまり変わらないと考えていたが，現在CAGR14%の割合で向上していることが分かった．
これは固定デバイスのコンテンツやアプリケーションの利用が日々増加しているため，それに伴って向上していると考えられる．
近い将来，固定デバイスがモバイルデバイスに全て取って代わられると思っていたため，このような速度で向上しているとは考えられなかった．

　次にデバイス数である．
モバイルデバイス数は2017年から2022年にかけて86億台から123億台に増加し，その内4億2200万台が5Gに対応するとあったが，大方は私の予想通りであった．
ただ，5Gの対応はもっと早いと思っていた．
約5%のモバイルデバイスしか5Gに対応しない理由としては，発展途上国もスマホは普及しているが，ネットワーク通信設備が完全に整っておらず，
先進国などの通信設備が整った場所だけが，優先的に5Gを導入できるのではないかと考える．
```

```
インシデント調査の方法の一つとして，メールヘッダの確認を行う手法がある．

ウイルス対策ソフトが，攻撃用と思われるメールを検知した場合，添付ファイルを閲覧したが，ウイルス対策ソフトが感染を防止し，
そのファイルを一時フォルダに格納することがある．
そこで，一時フォルダを調査し，不自然な実行ファイルを探索する．


②	ファイアウォール

外部のアドレスと思われるものがSrcに記録されていた場合，それに対応するDesが攻撃されていると考えることが出来る．
これにより，攻撃手法や攻撃元の割り出しが出来る可能性がある．

　また，単純に通信の発生時刻や通信プロトコルをログから確認できる．
それが妥当なものであるかどうかを判断し，妥当でないものは不正な通信が行われていると考えられる．

仮に80，443以外のポートへ通信している痕跡が発見された場合は，マルウェアに感染していると考えることができる．

　三つ目に，ボットに感染したPCにC&Cサーバへの通信を定期的に行い情報の取得を行わせる攻撃を受けているか調べる為に，
利用していないはずにも関わらず履歴に残っているURLを日ごとに集計し，不自然なアクセスが続いているものを抽出する方法がある．
もし不自然な連続アクセスが発見された場合は，ボットに感染していると疑われる．

④	DNSサーバ
```
